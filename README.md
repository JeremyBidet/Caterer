Projet Caterer
==============

Nom constituant le binôme
-------------------------

- BIDET Jérémy
- GALMICHE Mélody


Modalité de compilation
-----------------------

Pour compiler le programme, il est recommandé d'utiliser un IDE comme Eclipse.

En ligne de commande il est toujours possible de compiler.
Se placer dans le package racine fr.upem.jbidet.caterer et lancer la commande :
    $ javac Core/*.java DAO/*.java Solver/*.java Main.java


Execution du programme
----------------------

Il est recommandé d'utiliser encore un IDE (Eclipse) pour exécuter le programme.
Sélectionner le projet et lancer la commande Run.

Sinon, après compilation lancer la commande :
    $ java Main.class
là où le fichier MAin.class a été créé après compilation
